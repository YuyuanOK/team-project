# 待办事项应用

这是一个基于uni-app开发的待办事项应用，包含多种功能。

## 功能特性

- 任务管理：添加、编辑、删除和标记任务完成状态
- 按日期过滤：查看特定日期的任务
- 日历视图：通过日历界面选择日期
- 农历显示：在日历中显示农历日期和节日
- 头像上传：支持从相册选择或相机拍照上传头像
- 任务统计与分析：直观显示任务完成情况

## 农历日期计算

本项目使用纯本地算法计算农历日期和节日信息：

1. **农历编码表**：使用1900年至2100年的农历编码表进行精确计算
2. **支持闰月**：正确处理农历闰月情况
3. **农历节日**：支持显示春节、元宵节、端午节、七夕、中秋等传统节日
4. **二十四节气**：显示立春、雨水、惊蛰等24节气信息

这种纯本地算法不依赖网络API，保证应用在任何网络环境下都能正常显示农历信息。

## 开发与构建

1. 安装依赖
```
npm install
```

2. 开发预览
```
npm run dev:h5
```

3. 构建
```
npm run build:h5
```

## 技术栈

- uni-app框架
- Vue.js
- HTML5/CSS3
- ES6+

## 使用提示

1. 任务可以按类型和优先级进行筛选
2. 专注模式可以设置不同的时间长度
3. 长按任务项可以查看更多操作选项
4. 支持本地存储，数据不会丢失
